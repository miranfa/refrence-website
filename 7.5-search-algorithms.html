7.5 Search Algorithms

Objectives

To develop and write linear and binary search algorithms with guidance
To be able to describe the efficiency of an algorithm using "Big O" notation

Notes

Search Algorithms

Linear Search Algorithm

  1. Get array and target value
  2. Set a counter to i = 0
  3. Check if the element at arr[i] is equal to the target value
  4. If element matches, return index (i). Otherwise increment i by 1 and repeat step 3
  5. If the index is greater than the last indez of the array, return -1 to indicate the target value was not found

Binary Search Algorithm

  1. Sort array in ascending order
  2. Set the low index to 0 and high index to length - 1
  3. Set middle index to average of low and high
  4. If the middle index is the target, return the middle index
  5. If the target element is less than the element at the middle index, set the high index to the middle index -1
  6. If the target element is greater than the element at the middle index, set the low index to the middle index + 1
  7. Repeat steps 3-6 until low and high index are equal
  8. Return -1 if element is not found

Algorithm Efficiency

O(n) means that the maximum number of steps it will take to find and element in an array is equal to the number of elements in the array
